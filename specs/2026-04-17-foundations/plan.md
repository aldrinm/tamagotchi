# Plan - Foundations

## Task Groups

1. Discovery and design
   - Confirm Phase 0 scope from `specs/roadmap.md` and freeze non-goals for this phase.
   - Define baseline architecture for Angular 21 app shell, pet area rendering, and core domain models.
   - Resolve open questions around naming flow and placeholder UI expectations.

2. Implementation
   - Scaffold Angular 21 project and establish standard tooling/runtime scripts.
   - Build minimal app shell that renders pet area and placeholder values for vitals/state.
   - Implement foundational data structures and gameplay constants for `Pet`, `Stats`, and `State`.

3. Validation and release prep
   - Add or update tests that verify data model shapes and baseline rendering behavior.
   - Verify Phase 0 acceptance criteria: app compiles/runs and renders a pet UI with placeholder values.
   - Prepare PR notes that map implemented work back to roadmap Phase 0 deliverables.
